# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-02-13

### Changed

- Upgraded project version to `0.1.1`.
- Switched package manager from `npm` to `yarn`.
- Replaced `package-lock.json` with `yarn.lock`.

### Added

- Created `changelog.md` to track project history.

## [0.1.0] - 2025-12-28

### Added

- Added resume: `public/Resume-[RONiB].pdf`.
- Added new project screenshots in `public/`.
- Added `manifest.json`, `apple-icon.png`, and updated favicons/icons.
- Added `Error.jsx` component.

### Changed

- Migrated the entire codebase to **Next.js**.
- Optimized Home, About, Contact, and Projects pages for the Next.js App Router.
- Implemented backend API route for contact form submissions.
- Refined Header and Footer layouts.
- Updated project metadata and site configuration.

### Fixed

- Patched React Server Components CVE vulnerabilities.
- Resolved project cover image display issues.
- Fixed `onClick` event handling for improved navigation closing.
- Optimized performance and code readability across the application.
